# 🌾 AI-Powered Smart Farming Advisory System

## 📌 Project Overview

The **AI-Powered Smart Farming Advisory System** is a web-based application designed to assist farmers with intelligent decision-making. It provides crop recommendations, plant disease detection, and an AI chatbot for real-time assistance. The system integrates machine learning models and modern web technologies to deliver practical farming solutions.

---

## 🚀 Features

* 🌱 Crop Recommendation based on soil and environmental inputs
* 🦠 Plant Disease Detection using deep learning models
* 🤖 AI Chatbot for farming-related queries
* 🛒 Agro Products Marketplace for buying/selling products
* 👨‍🌾 Hire Labour module for connecting farmers and workers
* 🌍 Multi-language support (English, Hindi, Kannada)
* 🔐 User Authentication using Firebase
* 📊 Soil Test Analysis and recommendations

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap, JavaScript
* **Backend:** Flask (Python)
* **Database:** Firebase / MySQL
* **Machine Learning:** Scikit-learn, TensorFlow / PyTorch
* **Other Tools:** REST APIs, Git, GitHub

## ⚙️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/siddeshkr03/smart_farming_advisory_system.git
cd smart_farming_advisory_system
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Setup Firebase

* Add your Firebase credentials (do NOT upload secrets to GitHub)
* Place the key file locally or configure environment variables

### 5️⃣ Run the Application

```bash
python main.py
```

### 6️⃣ Open in Browser

```
http://127.0.0.1:5000/
```
